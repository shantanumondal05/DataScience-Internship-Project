# DataScience-Internship-Project
# Sales Data Analysis

# Project Overview
This project is a comprehensive data analysis of the Superstore Dataset. The goal was to analyze sales performance, identify trends, and determine the most profitable product categories using Python. 
The script performs data cleaning, calculates key performance indicators (KPIs), and generates visual insights to assist in business decision-making.

# Dataset
1. Source: [Kaggle - Sample Superstore Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
2. File Name:Superstore.csv
3. Description: Retail sales data containing transactions, dates, customer segments, and regional information.

# Key Features & Insights
The analysis covers the following business questions:
1.  Sales KPIs:Calculation of Total Sales and Average Order Value.
2.  Trend Analysis: Monthly sales tracking to identify seasonal peaks.
3.  Category Performance: Breakdown of sales by Furniture, Office Supplies, and Technology.
4.  Top Performers: Identification of the best-selling individual product and category.
5.  Data Quality:Handling encoding issues (`windows-1252`) and date parsing errors.

# Visualizations
1. Category Wise Sales: A bar chart comparing revenue across major product categories.
2. Monthly Sales Trend: A line chart illustrating sales performance over time.
3. Sales by Region: A pie chart representing the percentage contribution of each region to total sales.
4. Sales vs Profit: A scatter plot showing the relationship between sales and profit across transactions.
